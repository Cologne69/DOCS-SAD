analysis step - 1

Before making an application, analyzing is necessary to make a list of required features on the application. By analyzing (using Interview method), our current application SINAR TIMUR ERP has these features

With our secure Login & Register System, no one but only registered & logged in users can access the application, tested with penetrating tools, our application can withstand the penetration. Account information is stored in the database with hashed password.

Note
"anonymous user" cannot register a new user, instead, client need to be logged in first before registering a new user.

Our Product & Stock management uses MySQL database. With some magic (some triggers & procedures), our application can do

  • Decrease item stock when Sales to Customer & Return to Supplier is in progress
  • Increase item stock when Buying from Supplier & Return from Customer is in progress
  • Change product name, price, location where the product is being kept, and stock.
  • Auto-increment Transaction ID's

And features including:

  • Selling to customer receipt
  • Buying form supplier receipt
  • Transactions (incl. Selling & Buying) history
  • Returns history

Beside changing Product information, you can also change Customers & Suppliers information, such as, Customer's and Supplier's name, address, and telephone number.

Coding Step - 3

Yes, coding, the hard part. Every application goes on this part. The Back-end we chose Laravel framework. Why?

It is easy to implement, has built-in authentication system, has built-in filtering system, and many more built-in systems.

On top of Laravel we also chose Livewire as a fullstack Laravel Framework. Livewire is used to make real-time changes to the front-end (re-renders component) when a function or an API is called. This way, client does not need to refresh the page when a function or an API is called. 

Deploy Step - 4

Front-End done, Back-End Done, and now to deploy the application to the world so it can be accessed where ever the client is (as long as there is Internet connection available).

SINAR TIMUR ERP is hosted on Digital Ocean with Ubuntu 20.04 and Nginx (1 Virtual CPU, 2GB RAM). While the database (MySQL) is also hosted on Digital Ocean.
